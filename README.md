## Project Overview
This project focuses on building an interactive HR analytics dashboard using Power BI to analyze workforce data and derive actionable insights for HR decision-making.

## Key Objectives
- Monitor employee headcount, attrition, and retention trends
- Analyze workforce distribution by department, role, gender, and experience
- Identify high-attrition segments to support retention strategies

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV (data source)

## Key Metrics
- Total Employees
- Attrition Rate
- Average Tenure
- Gender Diversity Ratio
- Department-wise Attrition

## Features
- Interactive slicers (Department, Role, Gender, Experience)
- Drill-down analysis for attrition trends
- KPI cards and trend visuals for quick insights
- Clean and optimized data model
